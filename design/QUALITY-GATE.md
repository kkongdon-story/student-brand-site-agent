# Quality Gate

A site passes only when:

1. Brand identity is visible in the first viewport.
2. The reference is translated into concrete design decisions.
3. Image assets are generated, supplied, or explicitly queued with prompts.
4. Hover, click, and scroll interactions exist.
5. Mobile and desktop layouts are verified.
6. SEO/OG/GEO basics are present.
7. The student can explain which files to edit later.
