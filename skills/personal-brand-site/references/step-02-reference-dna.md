## Manual Mode

Use this when the student provides a landing page URL or screenshot.

Ask:

1. What part do you like most?
2. What should your site borrow strongly?
3. What must not be copied?
4. How strong should motion feel?
5. What image device is needed: character, photo, video, object, or abstract texture?

Translate the answer into:

| Dimension | Required Notes |
| --- | --- |
| Color | background, accent, contrast |
| Type | scale, weight, Korean/English mood |
| Spacing | dense, airy, editorial, grid |
| Layout | hero, sections, asymmetry, nav |
| CTA | wording, position, repeated moments |
| Image role | hero, mascot, proof, texture |
| Cursor | hover, magnetic, trail, none |
| Scroll | reveal, pin, parallax, section snap |
| Motion | subtle, expressive, playful, cinematic |
| Bring | what to adapt |
| Avoid | what would be imitation or mismatch |

## Reference Rule

The agent must not say "make it like this site." It must say exactly which visual and interaction decisions should be translated into the student's brand.
