# Student Guide

You will not start by coding.

1. Write your brand materials.
2. Pick a reference or let the agent suggest where to search.
3. Create an image brief.
4. Compile DESIGN.md.
5. Ask Codex to build the static site.
6. Run browser QA.
7. Publish with GitHub and Vercel.

If you do not know an answer, say "recommend it." The agent will suggest a beginner-safe default.
