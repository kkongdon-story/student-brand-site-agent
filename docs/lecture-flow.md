# Lecture Flow

## Before Class

- Codex or AI tool login
- GitHub login
- Vercel login

## 0-10 min

Show the finished example. Explain that the class teaches how to direct AI, not memorize code.

## 10-20 min

Stage 1: collect brand materials and one-line identity.

## 20-35 min

Stage 2: choose automatic or manual reference mode. Translate taste into reference DNA.

## 35-45 min

Stage 3: compile DESIGN.md. Emphasize that it is a contract.

## 45-55 min

Stage 4: build the static site and run browser QA.

## 55-60 min

Expansion: swap references, regenerate image direction, strengthen motion, prepare GitHub/Vercel publishing.
