# Reference DNA

## Mode

Manual

## Brand Direction Read From Stage 1

꽁돈은 AI를 어렵게 느끼는 사람에게 필요한 도구와 사고방식을 쉽게 찾아주는 강의/자료/협업 브랜드다. 홈페이지는 강의 신청, 협업 문의, 자료 탐색으로 연결되는 개인 브랜드 명함이어야 한다.

## Translation Table

| Dimension | Direction |
| --- | --- |
| Color | 밝은 종이 배경, 짙은 잉크색, 브라운/그린 포인트 |
| Type | 큰 한글 타이포그래피, 짧은 문장, 강한 첫인상 |
| Spacing | 넓은 여백과 불규칙하지만 읽히는 섹션 리듬 |
| Layout | 첫 화면에 캐릭터와 큰 카피를 함께 배치 |
| CTA | 카카오톡 상담 버튼을 hero와 footer에 반복 배치 |
| Component | sticky nav, keyword chips, proof rail, resource list |
| Image role | AI 안내자 캐릭터가 브랜드를 기억하게 함 |
| Cursor | 떠 있는 키워드와 버튼이 부드럽게 반응 |
| Scroll | 섹션이 하나씩 드러나는 스크롤텔링 |
| Motion | 장난스럽지만 과하지 않은 hover/click/scroll 반응 |

## Visual Archetype

Independent magazine + friendly AI workshop.

## Layout Archetype

Hero-as-poster, scrolling manifesto, resource gallery, contact footer.

## Bring

- 캐릭터 중심의 첫 화면
- 살아 있는 개인 창작자 느낌
- 마우스/스크롤 반응
- 밝고 친근한 편집 디자인

## Avoid

- 레퍼런스의 캐릭터를 그대로 모방
- 너무 어두운 AI SaaS 느낌
- 카드만 잔뜩 있는 일반 템플릿
- 반응 없는 정적 명함 페이지

## AI-Readable Materials

URL, screenshot path, color palette, typography description, component name, motion description, code idea를 함께 제공한다.

## DESIGN.md Language

꽁돈 사이트는 AI 안내자 캐릭터와 큰 한글 문장을 중심으로 시작한다. 방문자는 첫 화면에서 "어려운 AI를 내 일에 맞게 같이 찾아주는 사람"이라는 인상을 받아야 한다. 움직임은 hover, click, scroll reveal로 분명히 느껴져야 하지만 초보자가 수정 가능한 정적 파일 구조를 유지한다.

## Prompt Asset Summary

`prompt-pack.md`에는 design-first super prompt, hero/about/offer/resource/contact section prompts, hover/click/scroll interaction prompts, image prompts, revision prompts, and negative prompt를 저장한다.
