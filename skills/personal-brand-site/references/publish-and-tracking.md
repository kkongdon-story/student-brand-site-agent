# Publish and Tracking

The base class uses GitHub and Vercel.

## Required Publish Files

- `robots.txt`
- `sitemap.xml`
- `llms.txt`
- `README.md`

## Metadata

Check:

- Page title
- Meta description
- OG title
- OG description
- OG image placeholder or real asset
- Canonical URL placeholder
- JSON-LD
- Tracking code insertion point

## Student Explanation

GitHub stores the website files. Vercel turns that folder into a public URL.
